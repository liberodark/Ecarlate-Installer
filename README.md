# Morrowind - Ecarlate
Morrowind - Ecarlate Installer for Linux

# How to use :

Copy and Past in your terminal :

```bash
wget -Nnv https://raw.githubusercontent.com/liberodark/Ecarlate-Launcher/master/launcher.sh && chmod +x launcher.sh; ./launcher.sh
```

# Linux Compatibility :

- Arch Linux
- Ubuntu
- Linux Mint
- Debian
- Manaro
- Fedora
- Gentoo
