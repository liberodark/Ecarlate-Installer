# Morrowind - Ecarlate
Morrowind - Ecarlate Installer for Linux

# How to use :

Copy and Past in your terminal :

```bash
wget -Nnv https://raw.githubusercontent.com/liberodark/Ecarlate-Installer/master/install.sh && chmod +x install.sh; ./install.sh
```

# Linux Compatibility :

- Arch Linux
- Ubuntu
- Linux Mint
- Debian
- Manaro
- Fedora
- Gentoo
